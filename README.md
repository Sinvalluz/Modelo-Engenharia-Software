# 💰 Modelo-Engenharia-Software

Projeto desenvolvido para a disciplina de **Modelos, Métodos e Técnicas da Engenharia de Software** da Universidade Salvador (**UNIFACS**).

O sistema tem como objetivo auxiliar usuários no gerenciamento de despesas pessoais, permitindo controle financeiro de forma simples, intuitiva e organizada.

---

# 📌 Sobre o Projeto

O **Sistema de Controle de Despesas Pessoais (SCDP)** é uma aplicação web voltada para organização financeira pessoal.

A plataforma permitirá:

- Cadastro e autenticação de usuários
- Registro de receitas e despesas
- Controle financeiro por categorias
- Visualização de saldo
- Organização de lançamentos financeiros
- Relatórios financeiros
- Interface responsiva e intuitiva
